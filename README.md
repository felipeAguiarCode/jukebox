A mobile-first music app focused on **blues** and **jazz**. Browse moods, build playlists, queue tracks, and stream audio from YouTube—all with a clean, elegant interface.

---

## ✨ Features

- **Home** — Mood carousels with main categories (cards) and sub-moods (pills)
- **Playlist** — Track listing by mood with play and add-to-queue actions
- **Search** — Real-time search with sorting (A–Z / Z–A)
- **Queue** — Manage your queue with Play All and Reset
- **Now Playing** — Full player with play/pause, previous/next, seek controls
- **Now Playing Bar** — Mini player when browsing other screens
- **Toast notifications** — “Added to queue” feedback

---

## 🛠 Tech Stack

- **HTML5** — Semantic markup, data attributes
- **CSS3** — BEM methodology, CSS custom properties, mobile-first
- **JavaScript** — Vanilla JS, modular structure
- **Font** — [IBM Plex Sans](https://fonts.google.com/specimen/IBM+Plex+Sans)

## 🚀 Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/your-username/just-jazz.git
   cd just-jazz
   ```

2. Open `index.html` in a browser, or use a local server:
   ```bash
   npx serve .
   ```
